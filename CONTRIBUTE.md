Use `python 3.9.1`

## Install prerequisites

1. Create virtual env

```bash
python3 -m venv env && source env/bin/activate
```

1. Test and dev requirements

```bash
pip3 isntall -r requerements.txt && pip3 isntall -r requerements-dev.txt && pip3 isntall -r requerements-test.txt
```

After this steps all should works
